# Basic Types

All basic types are implicetely supported (including slices and vectors).

| Data Type                                             | Supported | Slice | Vector |
| ----------------------------------------------------- | --------- | ----- | ------ |
| bool                                                  | Yes       | Yes   | Yes    |
| Integer value: u8, u16, u32, u128, i8, i16, i32, i128 | Yes       | Yes   | Yes    |
| Float value: f32, f64                                 | Yes       | Yes   | Yes    |
